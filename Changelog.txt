v0.9.2.0
	General:
		Locusts are now obtained from chests just like vanilla
		Numbered Brane locust chests are now always AP Checks
		Max Locust Up's are included in the item pool and increase your maximum capacity for holding locusts
		Mon Bank now deals with Bonus Locusts, which are filler items not factored into logic
		
	Generation:
		Locustsanity is now forced on
		Generation shouldn't fail, probably
		
	Yaml:
		Removed Locustsanity option
		New setting "Max Locust Up Size"
		Greed Coin Amount range is now from 1-30
	
	Bugfixes:
		Receiving multiple items in quick succession will no longer delete locusts, progressive brands, or greed coins
		Burdens are now properly received from AP
	
	Misc:
		AP Items are now saved to file and should not reset upon game close or reconnecting to AP
		To properly reset between multiworlds, select End Run from the pause menu
		


v0.9.1.0
	General:
		New file "ap_room_name.csv" is now required
		
	Waypoints Menu:
		New AP Menu Page
		Three waypoint slots which can be set and then warped to at any time for convenience
		Stores locust count and floor, restoring both on warp
		Can only save waypoint on numbered branes
		Each brane has a unique WIP name/description
	
	Debug Menu:
		New AP Menu Page
		Displays several internal variables
		Should be helpful debugging
		If items aren't being received, are wiped, or you disconnect randomly, send a screenshot of the debug screen to the discord
	
	Connection Menu:
		Now raises errors and prevents connection if AP files are missing (instead of crashing)
		Input fields now allow strings longer than game width, up to 100 characters
		Added end-of-line indicator
	
	Tracker Menu:
		Added (temporary) counter for Greed Coins
	
	Bugfixes:
		End Run now resets (some) AP values
		
	Misc:
		Changed version number format
